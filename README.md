# Student Club Management System

The Student Club Management System is a database-driven application designed to manage and organize student club activities efficiently. The system helps administrators maintain records of students, clubs, events, memberships, and participation details in a centralized database.

This project was developed using SQL and database management concepts to demonstrate data storage, retrieval, and management through structured queries. The system aims to reduce manual record-keeping and improve the management of student club operations.

## Key Features
- Student Registration and Management
- Club Information Management
- Membership Tracking
- Event Management
- Database Reports and Queries
- Efficient Data Storage and Retrieval

## Technologies Used
- SQL
- Oracle Database / Oracle APEX
- PL/SQL

## Project Objective
The main objective of this project is to provide an organized and efficient way to manage student clubs, their members, and activities while applying database design and SQL concepts in a real-world scenario.
